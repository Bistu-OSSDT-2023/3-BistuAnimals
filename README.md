  FAQ  
  
    关于本项目的纯文本格式的常见问题解答。
    问：没有使用数据库是如何实现的存储用户姓名？
    答：由于在学习数据库连接数据库时出现了问题，我们就创建了对象数组
    来作为‘伪数据库’。当然这么使用是存在一些问题的，比如说当我第一次注册
    后可以登进论坛但是第二次进入论坛是仍需要再注册一遍。
